# ProJob Prototype UI

This is a standalone static prototype for the first ProJob app surface. It intentionally has no build step.

Open `index.html` directly in a browser, or serve this folder with any static file server. The CSS custom properties in `styles.css` mirror the root `../../design.md` tokens.

## Scope

The prototype demonstrates the initial `design.md` standard across:

- Field mobile job execution.
- Supervisor review.
- Office scheduling and queue management.

It is not the final frontend stack. The current architecture recommendation remains React + TypeScript + Tailwind + ProJob-owned components over shadcn/Radix primitives.
